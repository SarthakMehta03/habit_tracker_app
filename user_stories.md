# User Stories - Habit Tracker App

1. As a user, I want to sign up so that I can create an account.

2. As a user, I want to log in so that I can access my habits.

3. As a user, I want to add habits so that I can track my daily activities.

4. As a user, I want to view my habits on the home screen.

5. As a user, I want to open a detail screen to see more information about a habit.

6. As a user, I want my data to be stored locally so that it persists after restarting the app.

7. As a user, I want to fetch motivational quotes from an API.

8. As a user, I want to access settings to customize my experience.

9. As a user, I want to receive notifications as reminders to complete my habits.
